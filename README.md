- 호텔, 모텔 등 숙박업소 이용 시 
  대면 체크인/체크아웃으로 인해 발생되는 고객불편과      
  실물 열쇠 및 카드키 관리에 수반되는 불필요한 비용 개선   
- 호텔 예약 시스템과 호텔 운영이 이원화되어 발생하는 불필요한 인력 절감 및 관리 용이성 제공
